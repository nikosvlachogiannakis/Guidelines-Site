How to Use Lazygit to Improve Your Git Workflow

Install Lazygit

Follow the step-by-step guide in Install Lazygit.

How to Use Lazygit

Open your terminal and type:

lazygit

Lazygit UI is divided into six panels:

  • Status
  • Files
  • Branches
  • Commits
  • Stash
  • Previews

Status Panel

Shows an overview of your repository and current branch.

Files Panel

Shows modified, deleted, or unstaged files.

Branches Panel

Displays local and remote branches.

Commits Panel

Shows a list of commits in the current branch for viewing, checking out, or undoing.

Stashes Panel

Manage stashed changes to apply, drop, or view them.

Preview Panel

Preview unstaged changes, commits, logs, or file content.

Navigation Between Panels

Use left and right arrow keys or top panel keybindings:

  • 1 – Status
  • 2 – Files
  • 3 – Branches
  • 4 – Commits
  • 5 – Stash

Shortcuts and Key Mappings

Lazygit allows you to perform Git actions quickly with shortcuts:

  • a – Stage/unstage all files
  • space (Files panel) – Stage/unstage single file
  • c – Commit staged changes
  • P – Push commits
  • p – Pull changes
  • z – Undo last commit
  • s – Stash changes
  • S – View/apply stashes
  • n – Create a new branch
  • d – Delete a branch / discard changes
  • y – Copy file/commit info to clipboard
  • M – Merge selected branch
  • space (Branches panel) – Checkout branch
  • e – Open file externally
  • q – Quit Lazygit
  • ? – Open keybindings menu

How to Commit a File

  1. Select file with space, a, or double-click
  2. Press c to open commit panel
  3. Add commit message
  4. Press Enter

Push and Pull Code

  • P – Pull remote code
  • p – Push local commits

Create and Delete Branches

Create Branch

  1. Select Branches panel
  2. Press n
  3. Enter branch name
  4. Press Enter

Delete Branch

  1. Select Branches panel
  2. Press d
  3. Choose local or remote deletion

Note: Must be in Branches panel for n or d to work.

Undo a Commit

  1. Press z
  2. Select commit to undo
  3. Press Enter

Merge a Branch

  1. Press M
  2. Choose merge type (Merge, Squash Merge, Squash & Leave Uncommitted)
  3. Press Enter

Resolve Merge Conflicts

  1. Perform merge using M
  2. Conflicted files appear in Files panel
  3. Select file → press Enter → resolve conflicts with arrow keys and space

Discard Changes

Press d on a file or commit to discard.

Copy Information

  1. Select file or commit you want to copy from
  2. Press y to copy hash, path, or message

Other Keybindings

Press ? to open Lazygit keybinding help menu. Menu updates per panel context.
