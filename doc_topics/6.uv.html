Learn UV – Fast Python Package Manager

uv is a blazing-fast Python package and environment manager written in Rust by the creators of Ruff. It combines the roles of pip, virtual environments, and dependency managers like Poetry into a single modern tool.

Step-by-Step Summary

Step Action Command / Concept Description
1 Install uv curl -LsSf https://astral.sh/uv/install.sh | sh Installs uv
2 Create project uv init myapp Initialize a new project
3 Add dependencies uv add requests rich Adds packages and updates configuration
4 Run code uv run python src/myapp/main.py Runs code inside the environment
5 Create environment uv venv .venv Create standalone environment
6 Sync dependencies uv sync Install dependencies from lock file
7 Run tools uv run pytest Run tools inside the environment
8 Build package uv build Creates distribution files
9 Publish package uv publish Upload package to PyPI

Phase 1 – Getting Started

Install uv

macOS / Linux

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows (PowerShell)

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
Check installation: 
uv --version

Create a New Project

uv init myapp
cd myapp
Project structure: 
myapp/
├── pyproject.toml
├── src/myapp/
└── tests/

Add Dependencies

uv add requests rich
View dependency tree: 
uv tree

Run Code

Example Python file: 
import requests
from rich import print

def main():
    response = requests.get("https://api.github.com")
    print("Status:", response.status_code)

if __name__ == "__main__":
    main()
Run: 
uv run python src/myapp/main.py

Phase 2 – Environment Management

Create environment: 
uv venv .venv
Activate manually: 
source .venv/bin/activate
.venv\Scripts\activate
Sync dependencies: 
uv sync
Run commands inside environment: 
uv run pytest
uv run python src/myapp/main.py

Phase 3 – Packaging

Example configuration: 
[project]
name = "myapp"
version = "0.1.0"
description = "My uv project"
requires-python = ">=3.9"
dependencies = [
    "requests",
    "rich"
]
Build package: 
uv build
Generated files: 
dist/
├── myapp-0.1.0.tar.gz
└── myapp-0.1.0.whl
Publish package: 
uv publish

Advanced Topics

Install Python versions: 
uv python install 3.12
uv python use 3.12
Install global tools: 
uv tool install ruff
uv tool install black
Add dev dependencies: 
uv add --dev pytest
Remove dependency: 
uv remove requests
