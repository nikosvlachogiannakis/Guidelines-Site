# Developer Guidelines

A modern, interactive documentation website that provides structured guidelines, tools, and resources for developers. This project is designed to centralize essential workflows, tools, and references used in everyday development.

---

## 🚀 Overview

This site serves as a personal knowledge base and quick-access hub for:

* Development workflows
* Git and repository management
* Python environments and tooling
* Terminal utilities and productivity tools
* Cheatsheets and learning resources
* Curated external websites

The goal is to improve productivity, consistency, and efficiency in daily development tasks.

---

## 🧩 Features

* 📚 **Documentation Topics**
  Step-by-step guides for common development tasks such as Git operations, environment setup, Docker usage, and more.

* 🌐 **Curated Websites**
  Organized collection of useful tools and platforms (AI, courses, GitHub tools, etc.).

* ⚡ **Cheatsheets**
  Quick reference guides for commands, shortcuts, and concepts.

* 🎨 **Modern UI**
  Clean and responsive interface with animated backgrounds and interactive elements.

* 🔗 **Quick Access Cards**
  Easily navigate between tools and documentation sections.

---
